Modelling

This page is under contruction and the plots for ICON-SCM, ICON-CRM and ICON-LEM can change at a later stage due to adaptions in the experiment settings.
The plots show a comparison between observations and model forecast by ICON in its different configurations. ICON-D2 is an operational run in limited area mode with 2.2 km horizontal resolution with NWP physics. ICON-Global is an operational run on the globe.

The semi-idealistic simulations, ICON-SCM (Single Column Mode), ICON-CRM (Cloud Resolving Mode), and ICON-LEM (Large Eddy Mode) are driven by idealized 1D-forcings and run on a periodic torus grid. This has to be taken into account when comparing these two groups of simulations. While qualitative comparison is always possible between observations, realistic simulations (ICON-D2 and ICON-Global) and semi-idealistic simulations (ICON-SCM, ICON-CRM, and ICON-LEM), quantitative comparison is consistent only in each group (rows). See the bottom of this page for a detailed information of the shown data.

ICON configurations:
ICON-Global : NWP physics, activated dynamics, global grid
ICON-D2: NWP physics, activated dynamics, limited area grid with 2.2km horizontal resolution
ICON-SCM: NWP physics, de-activated dynamics, minimal torus grid (single column)
ICON-CRM-PER: NWP physics, activated dynamics, periodic torus grid (100x100 cells) with 2.5 km horizontal resolution
ICON-LEM-PER: LEM physics, activated dynamics, periodic torus grid (100x100 cells) with 100 m horizontal resolution
